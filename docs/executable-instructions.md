# FH6 Telemetry - Instruções de Configuração

## 1. Download do executável

> Link: https://drive.google.com/drive/folders/1t37vXzaScHZ33LO2zjW7BBJ9jbxwmjYm?usp=sharing

## 2. Configurar o jogo

No Forza Horizon 6, acesse **Settings > HUD e Jogabilidade > Telemetria** e preencha:

| Campo                         | Valor       |
| ----------------------------- | ----------- |
| Envio de Dados                | **Ligado**  |
| Endereço IP do Envio de Dados | `127.0.0.1` |
| Porta de IP do Envio de Dados | `20440`     |

## 3. Executar a aplicação

Dê dois cliques no arquivo `.exe` e abra no navegador:

```
http://localhost:3000/
```
